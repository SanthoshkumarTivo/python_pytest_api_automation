This repository contains a scalable API automation framework built using **Python, Pytest, and Requests**.  
It demonstrates real-world API testing practices including modular design, reusable utilities, and HTML reporting.

---

## Tech Stack
- Python
- Pytest
- Requests
- Pytest HTML Reports
- GitHub

---

## Framework Features
- Modular API client
- Centralized configuration
- Reusable test utilities
- HTML test execution reports
- CI/CD ready structure
